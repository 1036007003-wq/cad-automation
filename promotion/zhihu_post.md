# 如何用Python实现CAD图纸自动化处理？（开源工具分享）

大家好，我是建筑行业从业者，平时需要处理大量施工图纸。最近用Python写了一个**零成本的CAD自动化工具**，可以批量处理DXF图纸，特别适合「施工图转竣工图」这种重复工作。

## 🎯 解决什么痛点？

**手动处理的痛苦**：
- 每张图纸修改需要10-30分钟
- 容易漏改、改错
- 需要AutoCAD许可证（成本高）
- 难以批量处理

**使用自动化工具后**：
- 批量处理，每秒1张图纸
- 精确匹配，零错误
- 完全免费，开源
- 支持整个文件夹一键处理

## 🛠️ 工具功能

目前实现的功能（v0.1.0）：
1. ✅ **DXF文件解析** - 读取图层、图块、文字信息
2. ✅ **文字批量替换** - 如"施工图"→"竣工图"
3. ✅ **图层管理** - 批量冻结/解冻图层
4. ✅ **批量处理** - 整个文件夹一键处理

## 🚀 快速使用

### 安装
```bash
# 克隆仓库
git clone https://github.com/1036007003-wq/cad-automation.git
cd cad-automation

# 安装依赖
pip install -r requirements.txt
```

### 示例：施工图转竣工图
```bash
# 批量替换文字
python cli.py replace-text input.dxf output.dxf \
  --old "施工图" \
  --new "竣工图"

# 批量处理整个文件夹
python cli.py batch-process \
  ./施工图纸 \
  ./竣工图纸 \
  --text-replacements "施工图" "竣工图"
```
