# 【免费工具】CAD图纸批量处理，施工图转竣工图自动化

各位同行大家好！

分享一个**自己开发的CAD自动化工具**，专门解决「施工图转竣工图」这种批量处理需求。完全免费，开源，不需要AutoCAD许可证。

## 🎯 解决什么实际问题？

**大家有没有这种经历**：
- 手里几百张施工图纸，需要全部改成竣工图
- 每张图都要改文字、冻结图层、加竣工章
- 手动改一张要10-30分钟，还容易漏改、改错
- 公司AutoCAD许可证不够用，或者根本没预算买

**用这个工具后**：
- 批量处理，每秒能处理1张图纸
- 精确匹配，不会漏改或改错
- 完全免费，开源，不需要AutoCAD
- 支持整个文件夹一键处理

## ✅ 主要功能

1. **批量替换文字** - 比如把所有"施工图"改成"竣工图"
2. **图层管理** - 批量冻结/解冻图层（比如冻结"临时标注"）
3. **图纸信息查看** - 快速查看图层、图块、文字等信息
4. **批量处理** - 整个文件夹一键处理

## 🚀 怎么用？

### 第一步：下载工具

访问GitHub，下载到电脑：

（如果不会用GitHub，文末有详细教程）

### 第二步：安装Python依赖

```bash
# 安装依赖
pip install ezdxf click pyyaml
```

### 第三步：批量处理图纸

```bash
# 批量替换文字（"施工图" → "竣工图"）
python cli.py batch-process ./施工图纸 ./竣工图纸 --text-replacements "施工图" "竣工图"

# 批量冻结图层
python cli.py batch-process ./竣工图纸 ./输出 --freeze-layers "临时标注"
```

## 📁 支持什么格式？

**目前支持**：DXF格式（文本格式，大部分CAD软件都支持导出）  
**即将支持**：DWG格式（AutoCAD专有格式，下个版本支持）

**如果只有DWG文件**：
可以用AutoCAD或FreeCAD另存为DXF格式，或者用LibreDWG工具转换。
